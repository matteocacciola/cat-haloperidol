## Overview

Haloperidol is a powerful plugin for Grinning Cat that helps reduce hallucinations and ensures your Cat only replies
with information present in uploaded documents. This plugin uses structured prompting and fact-checking to validate
responses against your knowledge base.

## Install

Install the plugin directly from the Grinning Cat admin interface or clone this repository in the `cat/plugins` folder:

```bash
cd cat/plugins
git clone https://github.com/matteocacciola/cat-haloperidol.git
```
